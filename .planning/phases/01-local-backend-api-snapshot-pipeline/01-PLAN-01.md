# Plan 01-01: FastAPI Server Foundation & Snapshot REST Endpoints

**Phase:** 01 — Local Backend API & Snapshot Pipeline  
**Plan ID:** 01-01  
**Requirements:** REQ-INF-01, REQ-INF-02  
**Status:** Ready

## Plan Overview

Establish the core FastAPI backend server framework and implement REST endpoints for querying local snapshot files (`data/snapshot-*.json`).

---

## Tasks

<task id="1" type="tracer">

</action>
<verify>
Run `python -c "import fastapi, uvicorn; print('OK')"` or `pip install -r requirements.txt`.
</verify>
</task>

<task id="2" type="tracer">
<name>Implement coach/server.py with FastAPI endpoints</name>
<files>coach/server.py</files>
<action>
Create `coach/server.py` containing:
- FastAPI instance with CORS middleware (`CORSMiddleware` allowing `*` origins for local dev)
- `GET /api/health` returning `{"status": "ok", "version": "1.0.0"}`
- `GET /api/snapshots` listing available snapshot dates from `data/snapshot-*.json`
- `GET /api/snapshots/latest` returning the most recent snapshot payload
- `GET /api/snapshots/{date}` returning specific snapshot payload or 404 if missing
</action>
<verify>
Run unit tests in `tests/test_server.py`.
</verify>
</task>

<task id="3" type="tracer">
<name>Add dashboard command to main.py</name>
<files>main.py</files>
<action>
Update `main.py` CLI parser to add `dashboard` subcommand with `--host` (default `127.0.0.1`) and `--port` (default `8000`) arguments.
Add `cmd_dashboard()` function to invoke `uvicorn.run("coach.server:app", host=host, port=port)`.
</action>
<verify>
Run `python main.py dashboard --help` and verify parser output.
</verify>
</task>
