# Milestone Roadmap — wearcoach Dashboard & Analytics

## Overview

This roadmap details the execution phases to deliver a local interactive web dashboard and data science suite for `wearcoach`.

---

## Phases

### Phase 1: Local Backend API & Snapshot Pipeline
**Goal:** Establish local web server infrastructure (`python main.py dashboard`) with REST endpoints to serve snapshot data, activity logs, and coach interface.
**Requirements:** REQ-INF-01, REQ-INF-02  
**Plans:** 2 plans  
**Status:** Ready  

### Phase 2: Data Science & Analytics Engine
**Goal:** Build ACWR (Acute:Chronic Workload Ratio) training load calculations, strain/fatigue curves, and HRV readiness trend analysis algorithms.
**Requirements:** REQ-DS-01, REQ-DS-02  
**Plans:** 2 plans  
**Status:** Not Started  

### Phase 3: Interactive Dashboard UI & Dynamic Charting
**Goal:** Create a modern interactive web frontend featuring dynamic ````chart```` JSON block rendering, coaching chat interface, and analytics panels.
**Requirements:** REQ-UI-01, REQ-UI-02, REQ-UI-03  
**Plans:** 3 plans  
**Status:** Not Started  

### Phase 4: Multi-Sport Classification & Predictive Insights
**Goal:** Implement CrossFit / cross-training workload classification and race time / pace prediction models.
**Requirements:** REQ-DS-03, REQ-DS-04  
**Plans:** 2 plans  
**Status:** Not Started
